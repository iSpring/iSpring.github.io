# WebGlobe
基于HTML5原生WebGL实现的轻量级Google Earth三维地图引擎，支持诺基亚地图、微软Bing地图、腾讯地图、天地图、OpenStreetMap等。

没有使用第三方框架，无需插件，所有支持WebGL的浏览器均可使用。效率高，内存占用少。

会持续完善，目标是使其成为三维在线地图服务网站。
